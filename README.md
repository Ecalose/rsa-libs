# RSA libs

# 一、这是什么

RSA加密需要依赖的一些文件，在进行JavaScript逆向的时候经常会遇到一些RSA类型的加密，如果我们有一个RSA加密库就不必辛苦从目标网站的页面中扣那么多代码了，只需要扣除加密需要的参数然后引入这个库加密就可以了。
